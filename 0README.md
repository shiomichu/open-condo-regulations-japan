# 日本マンション管理規約オープンリポジトリ

**このリポジトリは、誰でも自由にフォークして利用・改変できます。**

国土交通省「マンション標準管理規約（単棟型）」をベースに、  
日本全国のマンションが自分たちの規約を作成・改善できる  
**オープンで共有可能な管理規約ハブ** を目指しています。

  
## このリポジトリの目的

このリポジトリは、国土交通省「マンション標準管理規約（単棟型）」をベースに、  
**日本全国のマンションが自由に利用・編集・派生できる “オープンな管理規約ハブ”** を目指して構築されています。

目的は以下のとおりです。

- 標準管理規約を **Markdown 化** し、誰でも読みやすく、差分管理しやすい形で公開する  
- 各マンションが **自分たちの規約を作るためのテンプレート** として利用できるようにする  
- GitHub の仕組み（フォーク・ブランチ・PR）を活用し、  
  **規約の改正履歴・議論・改善を透明に記録できる仕組み** を提供する  
- 全国の管理組合・理事会・管理会社・専門家が、  
  **共通のベース規約を共有し、改善し合えるコミュニティ基盤** をつくる  
- 将来的には、各マンションの規約を「派生版」として一覧化し、  
  **日本のマンション規約のオープンデータ化** を実現する

このリポジトリは、標準管理規約の単なる写しではなく、  
**「日本のマンション規約の中心（canonical repository）」** を目指すプロジェクトです。

## 利用ガイド（フォークして使う方法）

このリポジトリは、各マンションが自分たちの規約を作成・改正するために  
**フォークして自由に編集できること** を前提に設計されています。

### 1. リポジトリをフォークする
GitHub 上部の **Fork** ボタンを押して、自分のアカウントにコピーします。  
これがあなたのマンション専用の規約リポジトリになります。

### 2. 自分のマンション用に編集する
フォークしたリポジトリ内で、以下を自由に編集できます。

- 章ごとの Markdown ファイル  
- 別表（表形式の規約）  
- 申請書・承認書・誓約書などのテンプレート  
- README（マンション名や管理組合名に変更）

### 3. ブランチを切って作業する（推奨）
規約改正はブランチで行うと履歴が綺麗になります。


### 4. 改正内容をコミット・プッシュする
GitHub にプッシュすると、改正履歴が自動で残ります。  
理事会の議論や改正理由もコミットメッセージに残すと便利です。

### 5. 必要に応じて Pull Request を使う
理事会メンバーや管理会社と共同編集する場合は、  
Pull Request を使うことで **透明性のある議論** ができます。

### 6. ライセンスに従って自由に利用可能
このリポジトリはオープンライセンスで公開されているため、  
フォーク・改変・再配布が可能です（詳細は LICENSE を参照）。

---

## 推奨する使い方の例

- 自分のマンションの規約を **標準管理規約ベースで作り直す**  
- 理事会の改正案を **ブランチで管理し、PR で議論する**  
- 他マンションの規約を参考にする（派生版の比較）  
- 管理会社・専門家と **GitHub 上で共同編集** する  
- 改正履歴を **透明に残すことで、将来の理事会が困らない仕組み** を作る

---

## このリポジトリが目指す未来

- 全国のマンションがこのリポジトリをフォークして規約を作成  
- 派生規約が GitHub 上で一覧化され、比較・改善が容易になる  
- 標準管理規約の改正に合わせて、派生版も自動的にアップデートしやすくなる  
- 日本のマンション管理が **オープンで透明な知識基盤** の上に進化する
