# 別表第4　バルコニー等の専用使用権

本別表は、バルコニー・玄関扉・窓枠・窓ガラス・庭・屋上テラス等の  
「専用使用部分」と「専用使用権者」を示す。

---

## 1. 専用使用部分の区分

| 専用使用部分 | 位置 |
|--------------|------|
| バルコニー | 各住戸に接するバルコニー |
| 玄関扉 | 各住戸に付属する玄関扉 |
| 窓枠 | 各住戸に付属する窓枠 |
| 窓ガラス | 各住戸に付属する窓ガラス |
| 1階に面する庭 | 別添図のとおり |
| 屋上テラス | 別添図のとおり |

---

## 2. 専用使用権者

| 専用使用部分 | 専用使用権者 |
|--------------|--------------|
| バルコニー | 当該専有部分の区分所有者 |
| 玄関扉 | 同左 |
| 窓枠 | 同左 |
| 窓ガラス | 同左 |
| 1階に面する庭 | ○○号室住戸の区分所有者 |
| 屋上テラス | ○○号室住戸の区分所有者 |
