# 改版テンプレート（本文）

フォークしたマンションが独自の管理規約を作成・改版するための本文テンプレートです。

標準管理規約（本家）を直接編集しても構いませんが、  
このテンプレートを利用することで、独自規約を構造化しやすくなります。

- 第1章 総則  
- 第2章 管理組合  
- 第3章 管理者等  
- 第4章 管理組合の業務  
- 第5章 管理費等  
- 第6章 会計  
- 第7章 管理組合の会議  
- 第8章 役員  
- 第9章 専有部分の修繕等  
- 第10章 管理規約の変更  
- 第11章 雑則  

必要に応じて章を追加・削除・改変して利用してください。
